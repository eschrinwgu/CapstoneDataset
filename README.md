# CapstoneDataset
Dataset filled with images for use with my Capstone project. Taken from Kaggle; references present in the README


Data was downloaded from here, under an MIT license:
https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images/data

Sources Cited, as per Kaggle-published requirements:

Krizhevsky, A., & Hinton, G. (2009). Learning multiple layers of features from tiny images.

Bird, J.J. and Lotfi, A., 2024. CIFAKE: Image Classification and Explainable Identification of AI-Generated Synthetic Images. IEEE Access.

Real images are from Krizhevsky & Hinton (2009), fake images are from Bird & Lotfi (2024). The Bird & Lotfi study is available here.
